Github Pages简单搭建个人博客

发表于2020-10-05更新于2020-10-11字数统计193

一、新建仓库

在GitHub上新建仓库，repository name以username.github.io命名。

二、使用Hexo搭建个人博客

  • 使用 npm 安装 Hexo。
npm install -g hexo-cli
  • 新建项目
hexo init <项目文件夹>
  • 进入项目文件夹，执行
npm install

安装完依赖后，新建项目完成。

部署项目

执行以下命令，进行打包，

npm run build

完成后将public文件夹内的文件提交至第一步所建的仓库中。

然后在GitHub Settings中配置GitHub Pages的分支、目录即可。

本文只非常简单的介绍了部署个人博客的方法，所有操作基本均为手动，并未涉及到自动化编译等配置，具体步骤可参考hexo文档

本文使用 mdnice 排版
